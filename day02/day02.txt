HTML이란?
    : Hyper Text Markup Language

    Hyper Text
          (글,이미지...)
        :현재 페이지의 글 이미지를 초월하여 다른 웹 페이지의 자료들과 연결

    Markup Language
        : 웹 페이지 상의 컨텐츠를 어떻게 표현할지 명령하는 언어

    => 웹 페이지를 이동할 수 있게 해주고, 다른 웹 페이지의 이미지 등 컨텐츠에 접근
       가능하고 웹 페이지 내 컨텐츠를 제작하는 언어

    => 웹 페이지 때 기초 틀을 만들 때 쓰는 언어 ( 꾸미는 역할(style sheet)x )


    * 특징
        태그 형태를 띄고 있따는 특징
        Tag
        이 html 요소는 어떤 기능을 하고 있고 어떠한 속성을 가지고 있는지 나타낸다

        ex)
        - 컨텐츠가 있는 경우
        <시작 태그명 속성명="속성값"> ...(컨텐츠) </종료 태그명>

        - 컨텐츠가 없는 경우
        <태그명/>

        ex)
            <div id="menu_list" style="width: 300px"> ... </div>

            이 태그는 div이며 id 속성은 menu_list라는 값을 가지고 있고
            style 속성에는 width 300px이라는 값을 담고 있다.

    * html 기본 구조
        <html> -- 현재 해당 태그 안의 요소들이 html 언어를 사용한다
            <head></head> --- 해당 페이지의 정보, 창 이름, 검색 엔진 노출 정보, 링크
            <body></body> --- 사용자가 화면에 보여지는 화면, 실제 작업 장소
        </html> 

        확장자명은 .html

---------------------------------------------------------------------------------
            
    HTML 태그(element = 요소)의 종류

    <얖>
    <div> ... </div>  
        - 가장 사용 빈도가 높으며
        - 하나의 구역, 비어있는 박스, 틀

    <input/>
        - 텍스트 박스 태그
        - 글을 입력할 수도 있고, 체크박스, 버튼

        ex) 아이디, 비밀번호, 검색창 ...

    <p> ... </p>
        - 한 문장 단위의 글을 작성하는 태그
        - 위 아래 여백이 생기게 되는데 보통은 css로
          이러한 기존속성 값을 초기화 시키기 때문에 
          신경쓰지 않아도 됩니다.

        - p 태그 안에는 div 태그를 사용할 수 없다

    <span> ... </span>
        - 단어 혹은 특정 구간은 분간(지정)
        - 동그란 원을 만들 때 사용

    <img/>
    <video>
    <audio/>

        - 이미지, 비디오, 오디오와 같은 미디어 매체들을 위한 요소

    

    <a> ... </a>
        - 링크, 페이지 이동
        - href = "이동할 페이지의 경로"

        ex) 메뉴 클릭 시 다른 페이지로 이동(url 변동)

    <select>
        <option> ... </option>
        ....
    </select>

        - 드롭다운 메뉴 태그
        - option은 해당 드롭다운의 목록
          value라는 속성을 갖고 있음 선택된 option value가
          select 요소의 value가 된다.

          => 선택한 옵션이 select의 값이 된다.

        ex) 이메일 회원가입 할 때 이메일을 선택하는 드롭다운 메뉴

    <button> ... </button>
        - 버튼
        - 기본 type 속성 값은 "submit"
          <button type="button">... </button>

        - input type="button", "submit"로도 지정 가능

    <ul>
        <li> .. </li>
    </ul>

        - 리스트 태그
        - 메뉴 혹은 반복되는 아이템을 하나로 묶이 위해 사용


    **
        input과 button과 같이 특별한 기능을 하는 태그가 아니라면
        태그 명에 의미를 툴 필요는 없음

        단, 시멘틱 웹 태그는 제외한다

    
    input의 type속성

    input은 사용자가 입력 가능한 태그로서
    굉장히 많은 형태를 띄고 있습니다.

    ex) radio, button, submit, checkbox, text, password, email,
        number ....

    이러한 종류를 구별하기 위하여
    type이라는 속성을 통해
    해당 input이 어떠한 기능을 하고 있는 태그인지 명확히 명시

    그러나 type을 지정하지 않으면 default value는 text다


    <label> ... </label>
        - 묶어주는 태그
        - ex) 체크 박스 모양을 바꾸려할 때
              해당 체크박스를 보이지 않게 하는 css를 설정하고
              해당 자리에 label 태그를 두어
              마치 다른 이미지를 클릭해도 label과 연결된
              checkbox가 클릭되게 하는 태그
